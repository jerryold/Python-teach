# 03-5 Python的內建函式-enumerate


### enumerate()將一個可迭代物件的元素依序加入序號, 符合某些輸出需求. 



#### (1)程式範例
```python
# 建立一個list物件
data=['會資系', '財金系', '財稅系', '商務系', '企管系', '資管系', '應外系']

print(data)
print(type(data))
print('-'*30)

# 建立一個enumerate物件
print(enumerate(data))
print('-'*30) 

# 建立一個enumerate物件, 序號從1開始
for seq, name in enumerate(data, start=1):
    print('序號:', seq, '=> 名稱:', name)
    
print('-'*30) 
```

#### (2)程式範例 (enumerate內容只能取出一次)
```python
# 建立一個list物件
data=['會資系', '財金系', '財稅系', '商務系', '企管系', '資管系', '應外系']

print(data)
print(type(data))
print('-'*30)

# 建立一個enumerate物件
en=enumerate(data)

# 取出enumerate內容
for seq, name in en:
    print('序號:', seq, '=> 名稱:', name)
    
print('-'*30) 


# 取出enumerate內容(已無資料可取出)
for seq, name in en:
    print('序號:', seq, '=> 名稱:', name)
    
print('-'*30) 
```
