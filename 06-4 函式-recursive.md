# 06-4 函式-recursive


### 函式內的敘述呼叫函式自己, 稱為recursive.


#### (1)程式範例

```python
#------------------------------------
# 計算1*2*3*....*n的值
# 輸入:n
# 回傳:1*2*3*...*n
#------------------------------------
def fn(n):
    if n==1:
        return 1
    else:
        return n*fn(n-1)
#------------------------------------



#------------------------------------
# 主程式
#------------------------------------
print(fn(20))
#------------------------------------
```


執行結果:
```
2432902008176640000
```
