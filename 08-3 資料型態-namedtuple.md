# 08-3 資料型態(Data types)-namedtuple


### namedtuple可以設定其內元素的名稱, 作為Python內建資料型態tuple的另一種選擇.


#### (1)程式範例

```python
#--------------------------------------------------
# 匯入collections模組中的namedtuple類別
#--------------------------------------------------
from collections import namedtuple

#--------------------------------------------------
# 設定一個儲存成績的namedtuple 
#--------------------------------------------------
Scores = namedtuple('Scores', ['chi', 'eng', 'mat'])
s = Scores(65, 85, 75)  
print(s)
print('-'*30)

#--------------------------------------------------
# 以index取用namedtuple內容
#--------------------------------------------------
print('國文:{}'.format(s[0]))
print('英文:{}'.format(s[1]))
print('數學:{}'.format(s[2]))
print('-'*30)

#--------------------------------------------------
# 以名稱取用namedtuple內容
#--------------------------------------------------
print('國文:{}'.format(s.chi))
print('英文:{}'.format(s.eng))
print('數學:{}'.format(s.mat))
print('-'*30)
```


執行結果:
```
Scores(chi=65, eng=85, mat=75)
------------------------------
國文:65
英文:85
數學:75
------------------------------
國文:65
英文:85
數學:75
------------------------------
```



#### (2)程式範例

##### main.py

```python
#--------------------------------------------------
# 匯入collections模組中的namedtuple類別
#--------------------------------------------------
from collections import namedtuple

#--------------------------------------------------
# 設定一個儲存成績的namedtuple 
#--------------------------------------------------
Scores = namedtuple('Scores', ['chi', 'eng', 'mat'])

#--------------------------------------------------
# 讀入csv檔, 各筆資料存在namedtuple中
#--------------------------------------------------
import csv

for score in map(Scores._make, csv.reader(open('scores.csv', 'r'))):
    print('國文:{}  英文:{}  數學:{}'.format(score.chi, score.eng, score.mat))
    print('-'*30)
```

##### scores.csv

```python
80,60,75
65,85,90
70,50,85
80,90,85
100,90,85
```

執行結果:
```
國文:80  英文:60  數學:75
------------------------------
國文:65  英文:85  數學:90
------------------------------
國文:70  英文:50  數學:85
------------------------------
國文:80  英文:90  數學:85
------------------------------
國文:100  英文:90  數學:85
------------------------------
```
