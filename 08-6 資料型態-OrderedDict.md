# 08-6 資料型態(Data types)-OrderedDict


### 使用OrderedDict對儲存內容排序, 依鍵(key)或值(value)排序.


#### 程式範例

```python
#--------------------------------------------
# 匯入collections模組中的OrderedDict類別
#--------------------------------------------
from collections import OrderedDict

#--------------------------------------------
# 建立一個dict類別的實例
#--------------------------------------------
departments={'7':'應外系', '1':'會資系', '5':'企管系', '2':'財金系', '3':'財稅系', '6':'資管系', '4':'商務系'}

print(departments)
print('-'*80)

#--------------------------------------------
# 加入其他內容
#--------------------------------------------
departments['C']='數位多媒體設計系'
departments['B']='商品創意經營系'   
departments['A']='商業設計管理系'

print(departments)
print('-'*80)

#--------------------------------------------
# 由鍵(key)排序
#--------------------------------------------
d1=OrderedDict(sorted(departments.items(), key=lambda t: t[0]))
print(d1)
print('-'*80)

#--------------------------------------------
# 由值(value)排序
#--------------------------------------------
d2=OrderedDict(sorted(departments.items(), key=lambda t: t[1]))
print(d2)
print('-'*80)

#--------------------------------------------
# 由值(value)的長度排序
#--------------------------------------------
d3=OrderedDict(sorted(departments.items(), key=lambda t: len(t[1])))
print(d3)
print('-'*80)
```
