# 06-6 函式-使用套件(Packages)


### 將模組(1個或多個)存在一個獨立的資料夾中, 其內增加一個__init__.py檔案(一個沒有內容的空檔案), 該資料夾名稱(也許有多層)即是套件名稱. 其他程式可以用import指令匯入.

#### 架構範例
```
  |___ main.py
  |
  |___ <edu>
         |___ __init__.py  (init前後有2個底線)
         |___ ntub.py 
```
#### (1)程式範例

##### ntub.py
```python
#-------------------------------------------
# 一個計算傭金的函式
# 傳入:銷售金額, 職級(可不傳, 預設為'A')
# 回傳:傭金 
#-------------------------------------------
def commission(total, rank='A'):
    rate=0.0
    
    if rank=='A':
        if total>20000:
            rate=0.15
        elif total>10000:
            rate=0.1
        else:
            rate=0.08
    else:
        if total>20000:
            rate=0.12
        elif total>10000:
            rate=0.09
        else:
            rate=0.05        
        
    return int(total*rate)
```

##### main.py
```python
#--------------------------------------------------------------------------------
# 匯入edu套件中的ntub模組, 使用模組內commission函式時, 需用 edu.ntub.commission() 
#--------------------------------------------------------------------------------
import edu.ntub

tot=[(25000, 'A'), (36000, 'B'), (12000, 'B'), (9500, 'A'), (45000, 'A')]

for t, r in tot:
    print('銷售{}元, 職級{}, 傭金{}元'.format(t, r,  edu.ntub.commission(t, r))) 

print('-'*30)
```

執行結果:
```
銷售25000元, 職級A, 傭金3750元
銷售36000元, 職級B, 傭金4320元
銷售12000元, 職級B, 傭金1080元
銷售9500元, 職級A, 傭金760元
銷售45000元, 職級A, 傭金6750元
------------------------------
```


#### (2)程式範例(另外一種使用模組的方式)

##### main.py
```python
#---------------------------------------------------------------------------------------
# 從edu套件中匯入ntub模組中, 使用時模組中的commission函式時, 呼叫 ntub.commission() 即可.
#---------------------------------------------------------------------------------------
from edu import ntub

tot=[(25000, 'A'), (36000, 'B'), (12000, 'B'), (9500, 'A'), (45000, 'A')]

for t, r in tot:
    print('銷售{}元, 職級{}, 傭金{}元'.format(t, r,  ntub.commission(t, r)))

print('-'*30)
```
