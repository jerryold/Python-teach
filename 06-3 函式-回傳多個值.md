# 06-3 函式-回傳多個值


### 如有必要, 函式可以回傳多個值.


#### (1)程式範例

```python
#------------------------------------
# 將公分轉為呎, 吋
# 輸入:公分
# 回傳:呎, 吋
#------------------------------------
def cm_feet_and_inch(cm):
    feet, r=divmod(cm, 30.48)
    inch=r//2.54
    return feet, inch
#------------------------------------



#------------------------------------
# 主程式
#------------------------------------
f, i = cm_feet_and_inch(170)
print('{}公分 = {}呎{}吋'.format(170, int(f), int(i)))
#------------------------------------
```


執行結果:
```
170公分 = 5呎6吋
```
