# 08-5 資料型態(Data types)-Counter


### 使用Counter物件統計某個內容出現的次數.


#### 程式範例

```python
#--------------------------------------------
# 匯入collections模組中的Counter類別
#--------------------------------------------
from collections import Counter

#--------------------------------------------
# 建立一個Counter類別的實例
#--------------------------------------------
c = Counter()  

#--------------------------------------------
# 用map, list, tuple等iterable物件加入統計
#--------------------------------------------
c.update({'今天':1})
c.update({'天氣':1, '看起來':1})

c.update(['很', '好'])
c.update(['明天'])
c.update(['天氣', '很', '不', '好'])

c.update(('預料'))
c.update(('最近', '的', '天氣', '很', '不', '穩定'))

print(c)
print('-'*60)

#--------------------------------------------
# 排序
#--------------------------------------------
sorted(c.elements())
print(c)
print('-'*60)

#--------------------------------------------
# 列出數量最多的前n名(不必先排序)
#--------------------------------------------
print(c.most_common(2))
print('-'*60) 
```
