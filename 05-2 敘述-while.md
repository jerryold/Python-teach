# 05-2 敘述-while


### 用while進行重覆的工作. 迴圈內的敘述應該有相同的內縮.


#### (1)程式範例
```python
a=1

while a<=100:
    print(a)
    a=a+5       
```


執行結果:
```

96
```



#### (2)程式範例
```python
import random

while True:
    # 產生一個1~100的整數亂數
    a=random.randint(1, 100) 

    # 進行下一個迴圈
    if a==50:
        continue
    
    # 離開迴圈    
    if a==10:
        break    

    # 印出亂數
    print(a)           
```


執行結果:
```
52
44
17
27
63
29
84
80
61
44
19
....(略)
```
